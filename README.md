# Pizza Hut Sales Analysis using SQL

## Objective
This project analyzes Pizza Hut sales data using SQL to understand
order trends, revenue distribution, product performance, and
time-based customer behavior.

## Dataset Description
The dataset contains information related to:
- Orders and order timestamps
- Pizza types, categories, and sizes
- Quantity ordered and pricing

## Analysis Performed

### Basic Analysis
- Total number of orders placed
- Total revenue generated from pizza sales
- Identification of the highest-priced pizza
- Most commonly ordered pizza size
- Top 5 most ordered pizza types by quantity

### Intermediate Analysis
- Total quantity ordered per pizza category
- Order distribution by hour of the day
- Category-wise distribution of pizzas
- Average number of pizzas ordered per day
- Top 3 pizza types based on revenue

### Advanced Analysis
- Percentage contribution of each pizza type to total revenue
- Cumulative revenue analysis over time
- Top 3 pizza types by revenue within each category

## Tools & Skills Used
- MySQL
- SQL Joins, Aggregations
- Window Functions
- Date & Time Analysis

## Project Structure
- schema/ → Database table creation
- queries/ → SQL queries answering business questions
- insights/ → Business insights and recommendations
